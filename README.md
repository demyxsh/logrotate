# Usage
```
logrotate:
    container_name: logrotate
    image: demyx/logrotate
    restart: unless-stopped
    environment:
      DEMYX_INTERVAL
      DEMYX_PATH: /var/log/demyx
      TZ: America/Los_Angeles
    volumes:
      - /logs:/var/log/demyx
```

## logrotate.conf
```
${DEMYX_PATH}/*.log {
  su root root
  weekly
  missingok
  copytruncate
  rotate 52
  notifempty
  sharedscripts
}
```
